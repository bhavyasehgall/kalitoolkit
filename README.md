## 🚀 Kookieeyy Cyber Toolkit Suite

A modern GUI-based cybersecurity toolkit built with Python, designed to simplify reconnaissance, decoding, and steganography tasks through a unified interface.

## 🧠 Overview

Kookieeyy Cyber Toolkit Suite is a modular desktop application that brings together multiple security tools into a single, easy-to-use dashboard.

Instead of running commands manually in terminal, this toolkit provides a clean graphical interface to launch and manage tasks efficiently.

## ✨ Features

- 🖥️ Centralized Dashboard

    - Launch all tools from one place

- 🔍 Web Detection Module

     - Identify technologies used by websites
     - Perform basic reconnaissance

- 🔐 Decoder Tool

     - Decode multiple encoding formats

     - Useful for CTFs and security analysis

- 🕵️ Steganography GUI

     - Extract hidden data from images
     - Simplified interface for steghide operations

- ⚡ Fast & Lightweight

     - Minimal resource usage
     - Quick startup and execution

- 🧩 Modular Architecture

     - Easily extendable with new tools

### 🛠️ Built With

- Python 3

- Tkinter (GUI framework)

- Subprocess module (tool execution)
